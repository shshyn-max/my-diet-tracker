## 생활 아이템 웹앱 개발 by 상희
- 루틴트래커
  - 색상으로 루틴의 실행정도를 매일 트래킹 할 수 있는 기능 
- 타바타 타이머
  - 내가 정한 루틴으로 매일 돌릴 수 있는 타바타 타이머
  - 운동기록은 히스토리로 저장할 수 있도록 지원 
- 상품권 관리기
  - 카톡 등으로 받은 이미지 상품권의 잔액을 계산하도록 돕는 앱
  - 사용한 상품권의 종류, 금액 등을 통계로 볼 수 있도록 제공 (커밍순~) 
- 투두리스트 관리기
  - 아직은 흔한 투두리스트 관리기
  - 심플하게
  - 집안일 투두리스트로 포커싱 할 예정
